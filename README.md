# meuprimeiro-repositorio
# 🧮 Calculadora Java

Projeto desenvolvido durante meus estudos em **Java** com o objetivo de praticar lógica de programação, organização de código, Git e GitHub.

A calculadora foi construída de forma modular, permitindo que novas operações e novas interfaces sejam adicionadas no futuro sem comprometer a estrutura do projeto.

---

# 🚀 Funcionalidades

Atualmente o projeto possui:

* ✅ Soma
* ✅ Subtração
* ✅ Multiplicação
* ✅ Divisão
* ✅ Execução por argumentos (CMD)
* ✅ Execução por Scanner (IntelliJ/Terminal)
* ✅ Estrutura modular utilizando classes separadas para cada operação

---

# ▶ Como executar

## Pelo IntelliJ IDEA

1. Execute o projeto.
2. Escolha o modo **Scanner**.
3. Informe a operação desejada.
4. Digite os números solicitados.

---

## Pelo Prompt de Comando (CMD)

Exemplo:

```bash
java my.company.calculator.codigo_main.Calculadora2 10 20
```

Depois selecione a operação desejada.
